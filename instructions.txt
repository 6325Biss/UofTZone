### Setup instructions
## (must install mongodb first.)

## Install all modules
$ npm install

## Start a mongo server
$ mkdir mongo-data
$ mongod --dbpath ./mongo-data

## Set up the default data, follow the guide below in order, do not change the order.
## open another terminal window and type these commands:
1. node mongo-add-user.js
2. node mongo-add-admin.js
3. node mongo-add-post.js
4. node update-defaults.js
